# AHID
Hardware based, AI assisted, Automated Human Input Device using a Raspberry Pi Zero and the USB Gadget Kernel module.

# Use Cases
  - Undetectable cheats; e.g. aimbot, auto farming, etc.
  - Mimic repetative user interaction.
